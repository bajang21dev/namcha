# Basic Node.js with ECMAScript 2015 (ES6)

This application for a beginner need to learn about node.js and es6.

Libraries integrated:
- ECMAScript 2015 (ES6)
- Express.js 4
- MongoDB by Mongoose
- Stylus css engine
- Mocha unit testing

Recommended:
- Node.js 4 or higher

Preparing:
- Start MongoDB (Run command "mongod" on your terminal)
